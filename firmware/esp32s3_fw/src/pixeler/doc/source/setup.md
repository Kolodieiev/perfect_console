# Налаштування Pixeler

Для зручності усі доступні налаштування Pixeler зібрано в заголовкових файлах в каталозі `pixeler/setup`.
Рекомендується уважно ознайомитись з цими файлами.
Редагуючи їх вміст, можна підключати/відключати функціонал, та налаштовувати параметри периферії в залежності від задачі.
Якщо в проєкті не використовується перефирія, налаштування якої є в файлах налаштувань, на ці налаштування можна не зважати і залишити як є.
Піни, частоти, тощо не будуть налаштовані статично, тобто ці налаштування не вплинуть ні на що.

::: {important}
Деякий функціонал, наприклад GUI, є глобальним та буде збиратися з прошивкою, якщо його не вимкнено через відповідний прапор.
Тому варто читати коментарі біля визначень в файлах налаштувань.
:::

Більш детально вміст деяких файлів буде описаний у розділах документації, присвячених відповідному функціоналу.
