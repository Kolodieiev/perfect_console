# Налаштування середовища розробки

Для збірки прошивки на Pixeler, необхідно першим кроком встановити VS Code.<br>
Далі необхідно встановити venv командою

    sudo apt install python3-venv

Та утиліти для роботи з архівами 

    sudo apt install xz-utils liblzma-dev

Після цього знайти в менеджері розширень Pioarduino та встановити його.
Зверніть увагу, не Platformio, а саме Pioarduino.
Pioarduino це форк Platformio, що підтримує останні версії Arduino та ESP-IDF.

::: {important}
Обидва розширення використовують папку .platformio, тому можуть перезаписувати критичні файли одне одному та ламати збірку.
Найкраще до встановлення Pioarduino, якщо Platformio активно використовується для збірки інших проєктів, зробити резервну копію каталогу
та відключити розширення Platformio задля уникнення конфліктів.
:::
